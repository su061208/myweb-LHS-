PROJECT 01

SolidWorks Automata 설계

캠 메커니즘을 적용한 오토마타의 전 과정을 설계하고, 동역학 시뮬레이션을 통해 기계적 무결점을 검증했습니다.

[SO WHAT? / 트러블슈팅 성과]

초기 설계 시 캠의 접촉각 오류로 인한 급격한 마모 문제를 발견. 동역학 해석 프로그램을 통해 최적의 압력각을 재도출하여 동력 전달 효율을 15% 개선하고 기구부 내구성을 1.5배 증대시켰습니다.

PROJECT 02

EV3 실시간 로그 수집 시스템

다양한 환경 변수를 실시간으로 로깅하고 모니터링하여, 하드웨어 오류를 사전에 포착하는 시스템입니다.

[SO WHAT? / 트러블슈팅 성과]

데이터 샘플링 중 인터럽트 충돌로 인한 데이터 유실 현상(8% 손실)을 포착. 멀티태스킹 우선순위 재스케줄링을 통해 데이터 누락률 0%를 달성하며 시스템 무결점을 증명했습니다.
